##### GENERAL WORKFLOW TO COMPILE AND INSTALL DDX #######

##### install packages:

libdc1394-22-dev
lib11-dev
libsdl1.2-dev
libreadline5-dev
python2.5-dev
python-all-dev (optional but removes necessity to link configs (see below))
libxmu-dev
manpages-de-dev (optional)

##### general workflow:

(unpack)

$ ./configure
$ make
$ make install

##### if not working, link:

$ ln -s python2.5-config python-config
$ ln -s /usr/local/lib/* /usr/lib
